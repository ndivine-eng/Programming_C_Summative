# Project 4 - Data Analysis Toolkit (Function Pointers + Callbacks)

## Features implemented
- Function pointer dispatcher (menu option -> function pointer)
- Callback-based filtering, transformation, and sorting comparison
- Dataset operations: sum/average, min/max, filter, transform, sort, search
- Dynamic dataset memory (`malloc`, `realloc`, `free`)
- Load/save dataset from file (`dataset.txt`)
- Input validation and NULL callback checks

## Compile and run
```bash
gcc -Wall -Wextra -std=c11 data_toolkit.c -o data_toolkit
./data_toolkit
```
