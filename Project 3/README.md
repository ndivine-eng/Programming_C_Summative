# Course Performance and Academic Records Analyzer in C

## Overview
This project is a C program for managing academic records. It supports adding, updating, deleting, displaying, searching, sorting, saving, and loading student records.

## Features
- Dynamic memory allocation using malloc/realloc/free
- Structures for student data
- File handling with text file storage
- Search by ID or name
- Sort by ID, name, or GPA
- GPA analysis and reporting
