---

##  Project 3 — Academic Records Analyzer
Manages student records dynamically and performs academic analysis.

### Features
- CRUD operations
- GPA calculation
- Sorting and searching
- Reports (average, highest, lowest, median GPA)
- File persistence (records.txt)

### Run
gcc -Wall -Wextra -std=c11 academic_records.c -o academic_records
./academic_records

---

##  Project 4 — Data Analysis Toolkit
Processes numerical datasets using function pointers and callbacks.

### Features
- Dynamic dataset
- Filtering, transforming, sorting
- Sum, average, min, max
- File handling

### Run
gcc -Wall -Wextra -std=c11 data_toolkit.c -o data_toolkit
./data_toolkit
